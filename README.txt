------------------------
Development Environment
------------------------
Java: JDK 7
IDE: Netbeans 8
Build tool: Maven 3
Junit: 4.10




---------------------------
Run Instruction
---------------------------
1. Install Java and set Java environment 

2. Unzip Hans_Li.zip

3. From command line, go to dist folder of the above unzipped file and run command:

java -classpath PrintCalculator-1.0-SNAPSHOT.jar com.printcalculator.PrintJobRunner

4. Enter job file path:
Please enter your input print job file path:
e.g. 
C:\Users\lih\Documents\NetBeansProjects\PrintCalculator\sample_data\printjobs.csv
